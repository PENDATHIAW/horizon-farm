# Horizon Farm Web

Site institutionnel premium Horizon Farm.

## Déploiement

Dans Vercel, choisir comme Root Directory :

```txt
sites/horizon-farm-web
```

Commandes :

```txt
npm install
npm run build
```

Le dossier investisseur PDF est référencé depuis le fichier public déjà présent dans le repo principal après intégration du dossier investisseur.
