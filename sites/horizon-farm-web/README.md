# Horizon Farm — Site institutionnel

Site vitrine premium pour **Horizon Farm** — investor-ready, multilingue FR/EN, dark/light mode.

## Démarrage

```bash
npm install
npm run dev
```

## Déploiement Vercel

Importer le repo, root directory : `sites/horizon-farm-web`

| Paramètre | Valeur |
|-----------|--------|
| Build | `npm run build` |
| Install | `npm install` |
| Output | `dist` |

## Fonctionnalités

- Hero vidéo cinématique
- 4 business units + section Tallow & Go
- Traçabilité interactive « Farm to Skin »
- Section investisseurs (PDF dossier)
- FR / EN · Dark / Light mode
- Mobile-first · SEO meta tags
